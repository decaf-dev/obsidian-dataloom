# Plugin settings

Plugin settings can be accessed from **Settings > DataLoom**. They are applied for all looms in a vault.

| Name                                   | Value          |
| -------------------------------------- | -------------- |
| Create looms in the attachments folder | `<true/false>` |
| Default location for new looms         | `<path>`       |
| Export content as markdown             | `<true/false>` |
| Default embedded loom width            | `<width>`      |
| Default embedded loom height           | `<height>`     |
