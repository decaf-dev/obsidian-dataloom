# Number cell

## Description

This cell renders a number.

## Usage

This cell accepts any number. A number is a positive or negative integer, or decimal.

## Number format

A number can be displayed in the `Number` or `Currency` format. When a number is displayed as a number, it will render any additional formatting options. When a number is displayed as currency, it will be prefixed with the currency symbol.
