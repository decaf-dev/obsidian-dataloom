# Quick start

## Installation

### Install the plugin

1. In Obsidian, open **Settings**
2. Go to **Community plugins**
3. Select **Browse**
4. Search for **DataLoom** by **DecafDev**
5. Select **Install**
6. Then select **Enable**

<VideoEmbed url="https://dataloom.b-cdn.net/plugin-installation.mov" />

### Configure Obsidian

DataLoom saves data in JSON format in `.loom` files. Some changes are required to Obsidian's default settings in order to allow the plugin to work properly.

#### Enable detection of loom files

By default, Obsidian won't display `.loom` files in the suggest menu that opens when you type double brackets `[[`. To enable this behavior, you must turn on the detection of all file extensions.

1. In Obsidian, open **Settings**
2. Select **File & Links**
3. Toggle **Detect all file extensions**

<VideoEmbed url="https://dataloom.b-cdn.net/enable-file-extensions.mov" />

#### Enable syncing of loom files

By default, Obsidian won't sync `.loom` files between devices. To enable this behavior, you must turn on sync for all file extensions.

1. In Obsidian, open **Settings**
2. Select **Sync**
3. Choose and connect to a remote vault
4. In the remote vault settings, Toggle **Sync all other types**
5. Repeat these steps on all devices that you want to sync `.loom` files to

<VideoEmbed url="https://dataloom.b-cdn.net/enable-sync.mov" />

## Create a loom

There are 4 ways to create a loom

### Sidebar icon

Click the table icon in the sidebar

<VideoEmbed url="https://dataloom.b-cdn.net/create-loom-sidebar.mov" />

## Right click on a folder

Right-click on any folder and select **New loom**

<VideoEmbed url="https://dataloom.b-cdn.net/create-loom-folder.mov" />

## Command palette

Open the [command palette](https://help.obsidian.md/Plugins/Command+palette) and search for **Create loom**

<VideoEmbed url="https://dataloom.b-cdn.net/create-loom-command-palette.mov" />

### Embed into note

You can also embed a new loom into an existing note. Open a markdown note and move
the cursor to the line where you wish to insert a loom. Open the command palette
and search for **Create loom and embed into current file**

<VideoEmbed url="https://dataloom.b-cdn.net/create-loom-command-palette-embed.mov" />
