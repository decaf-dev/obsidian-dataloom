# Date cell

## Description

This cell renders a date.

## Usage

Select a date cell to open the date cell edit menu. Type a date value. Press `Enter` or `return` for it to be saved. If the value is invalid, a red outline will appear around the input. Enter a valid value and press `Enter` or `return` for the value to be saved.

To close the menu without saving, click outside of the menu or press `esc`.

### Date format

The date format is how the date will be displayed. Changing the date format for a cell will change the date format for all cells in the same column.

You may clear the date by pressing the `clear` button.
