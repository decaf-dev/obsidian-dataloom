# File cell

## Description

This cell renders an Obsidian link.

## Usage

When the menu opens, a list of vault notes will appear. Type any value to filter the list of notes. Notes are filtered using the fuzzy sort algorithm. You may use the `up` and `down` arrow keys to navigate the list. Click an item or press `Enter` or `return` to select the highlighted value. The value you choose will be saved.
