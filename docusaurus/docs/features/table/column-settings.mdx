# Column settings

## Wrap content

If enabled, the content of a cell will wrap instead of being cut off when the cell width is smaller than the content width.

## Date format

The format in which a [unix time](https://en.wikipedia.org/wiki/Unix_time) value should be displayed.

There are 5 different formats

-   `Month/Day/Year` - e.g. 05/01/2023

-   `Day/Month/Year` - e.g. 01/05/2023

-   `Year/Month/Day` - e.g. 2023/05/01

-   `Full` - e.g. May 1, 2023

-   `Relative` - e.g. Yesterday 5:36 pm

## Number format

A number can be formatted as a number or as a currency.

### Number

When `Number` is selected, you will have options to change the `prefix`, `suffix`, and `separator`

-   Prefix - What should be displayed before the number. e.g. `$`
-   Suffix - What should be displayed after the number. e.g. `%`
-   Separator - What should be used to separate thousands. e.g. `,`
