# Calculating column values

Calculations can be preformed on all the cells in a column. Click the `Calculate` button at the bottom of a column. A menu will appear with a list of calculations that can be preformed on the column.

## General Calculations

-   None - Do nothing
-   Count all - The total number of cells
-   Count values - The total number of values. This count each tag as a single value.
-   Count unique - The total number of unique values
-   Count empty - The number of empty cells
-   Count not empty - The number of non-empty cells
-   Percent empty - The percentage of empty cells
-   Percent not empty - The percentage of non-empty cells

## Numeric Calculations

Numeric calculations can only be preformed on a column set to type `Number`

-   Sum
-   Average
-   Min
-   Max
-   Median
-   Range
